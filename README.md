 AI Workforce Displacement Analysis

This project explores global AI workforce displacement trends from 2020 to 2026 using a dataset of country, sector, and AI adoption metrics.

## Contents

- `AI.ipynb`: Jupyter notebook with data loading, cleaning, and visualization steps.
- `ai_workforce_displacement_global_2020_2026.csv`: Dataset used for analysis.
- `output4.png`, `output5.png`, `output6.png`, `outputAI.png`: Generated output visuals from the notebook.
- `project.py`: placeholder file (currently empty).
- # What it does

The notebook performs:

- dataset inspection and summary statistics
- missing value checks
- formatting and column type adjustments
- correlation heatmap for AI and workforce metrics
- ranking of industry sectors by average workforce displacement
- trend analysis of AI adoption and workforce displacement across income groups
- scatterplot comparing automation risk and workforce displacement

## How to run

1. Open `AI.ipynb` in Jupyter or VS Code.
2. Make sure `ai_workforce_displacement_gglobal_2020_2026.csv` is in the same folder.
3. Run the notebook cells to generate plots and inspect results.
